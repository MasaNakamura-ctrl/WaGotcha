## テスト実施
### DB立ち上げ
```
docker comopose up -d
```
### Vue立ち上げ
```
npm run dev
```
### Spring立ち上げ
```
./mvnw spring-boot:run // macOs
```
### 画面表示
localhost:5173を表示
![表示画面](./img/no1.png)

### DB停止
```
docker comopose down
```
